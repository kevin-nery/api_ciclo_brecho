# API Ciclo Brechó

O Ciclo Brechó é um projeto que estou fazendo para o gerenciamento de um brechó. Ele surgiu a partir da ideia da minha namorada de criar um brechó, onde ela pediu minha ajuda para montar uma planilha para controlar os produtos.

Estou usando o projeto para praticar a criação de API's principalmente, tendo em vista que meu foco de aprendizado é o back-end.

## Funcionalidades

Além do CRUD básico de produtos que já está pronto, estou integrando com o Cloudinary para o envio e armazenamento de fotos, o qual meu banco já está preparado. Atualmente o projeto está com o inicio da integração feito, a conclusão é o meu próximo passo.

## Tecnlogias

Estou fazendo o back-end com JavaScript, Prisma ORM para manipulação do banco, express para a criação do servidor e gerenciamento de rotas. Como banco, tenho uma imagem do Postgres Alpine rodando em um docker.
